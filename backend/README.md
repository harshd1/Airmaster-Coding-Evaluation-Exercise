# Backend API

This folder contains the C# ASP.NET Core backend implementation for the Airmaster order management platform.

## Run

1. Open in Visual Studio or VS Code.
2. Restore NuGet packages.
3. Run the application.

## Endpoints

- `GET /api/products`
- `GET /api/products/{id}`
- `GET /api/orders/{orderId}`
- `POST /api/orders`
- `POST /api/payments`
